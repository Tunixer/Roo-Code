### 支持的命令

| 命令                  | 参数         | 描述                 |
| --------------------- | ------------ | -------------------- |
| `enable`              | 无           | 启用机械臂           |
| `disable`             | 无           | 禁用机械臂           |
| `home`                | `HomePose`   | 移动到Home位置       |
| `move_to_target`      | `TargetPose` | 移动到目标位置       |
| `stop`                | 无           | 停止当前运动         |
| `emergency_stop`      | 无           | 急停                 |
| `reset`               | 无           | 复位错误状态         |
| `save_home_position`  | `HomePose`   | 保存Home位置设置     |
| `reset_home_position` | 无           | 重置Home位置为默认值 |

### 🎯 核心功能

- **笛卡尔位置控制**: 支持X、Y、Z轴精确位置调节
- **RPY角度控制**: 支持Roll、Pitch、Yaw三轴姿态调节
- **实时状态显示**: 显示当前位置、姿态和关节状态
- **多步长调节**: 提供小、中、大三种步长选项
- **安全控制**: 包含急停、复位、启用/禁用等安全功能
- **自定义Home位置**: 支持用户手工设置和保存Home位置

## Home位置设置功能

### 🏠 功能概述

Home位置设置功能允许用户自定义机械臂的原点位置，包括XYZ坐标和RPY角度。这个功能对于不同的工作场景和应用非常有用。

### 🔧 使用方法

#### 1. 打开Home位置设置

- 点击"回到原点"按钮旁边的⚙️设置按钮
- 设置面板将展开显示当前Home位置和控制界面

#### 2. 调节Home位置

- **滑块调节**: 使用滑块连续调节各轴位置
- **数值显示**: 实时显示当前设置的Home位置数值
- **范围限制**:
    - X/Y轴: -500mm 到 +500mm
    - Z轴: 0mm 到 800mm
    - Roll/Yaw: -180° 到 +180°
    - Pitch: -90° 到 +90°

#### 3. 保存和管理

- **保存Home位置**: 将当前设置保存为新的Home位置
- **设置为当前位置**: 一键将机械臂当前位置设为Home位置
- **重置为默认**: 恢复到系统默认的Home位置 (0,0,400,0,0,0)

### 📋 默认Home位置

```typescript
{
  x: 0,      // X轴位置 (mm)
  y: 0,      // Y轴位置 (mm)
  z: 400,    // Z轴位置 (mm)
  roll: 0,   // 横滚角 (度)
  pitch: 0,  // 俯仰角 (度)
  yaw: 0     // 偏航角 (度)
}
```

### 💡 使用建议

1. **安全第一**: 设置Home位置时确保该位置是安全的，不会与工作环境发生碰撞
2. **工作流程**: 根据具体应用场景设置合适的Home位置，提高工作效率
3. **备份设置**: 重要的Home位置设置建议记录备份
4. **测试验证**: 设置新的Home位置后，先进行低速测试确认安全性

### 🔄 工作流程示例

1. 手动调节机械臂到理想的起始位置
2. 点击"设置为当前位置"按钮
3. 点击"保存Home位置"确认设置
4. 使用"回到原点"功能验证设置是否正确
